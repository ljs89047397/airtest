## SEQ-01: EMP 라이프사이클 — DRAFT → APRVD (3 actor 전환)

```mermaid
sequenceDiagram
    autonumber
    actor AIR as 항공사(kal_user)
    actor KOT as KOTSA(kotsa01)
    actor MOL as MOLIT(admin01)
    participant W as Browser
    participant C as EmpPlanController
    participant S as EmpPlanService
    participant A as AuthorityInterceptor
    participant DB as PostgreSQL

    AIR->>W: 로그인 + EMP 신규 작성
    W->>C: POST /api/emp/plan
    C->>A: 권한 검증 (URL prefix)
    A-->>C: OK
    C->>S: createEmpPlan(vo, user)
    S->>DB: INSERT tn_emp_plan (DRAFT)
    DB-->>S: empPlanId=EP0001
    S-->>C: 200 OK
    C-->>W: empPlanId
    W-->>AIR: 상세 화면 (7탭 입력)

    AIR->>W: 제출 버튼
    W->>C: POST /api/emp/plan/{id}/submit
    C->>S: submit()
    S->>DB: UPDATE emp_st_cd=SBMTD + th_emp_chg_hstry
    DB-->>S: 1 row
    S-->>W: 200

    Note over KOT,MOL: 결재함 알림 → KOTSA 검토 진입

    KOT->>W: 로그인 + EMP 상세 진입
    W->>C: POST /api/emp/plan/{id}/review
    C->>S: review() — DRAFT가 아닌 SBMTD만 가능
    S->>DB: UPDATE emp_st_cd=RVWNG
    DB-->>S: OK
    KOT->>W: 권고 버튼
    W->>C: POST /api/emp/plan/{id}/recommend
    S->>DB: UPDATE emp_st_cd=RCMDD

    MOL->>W: 결재함 > 승인
    W->>C: POST /api/emp/plan/{id}/approve
    C->>S: approve()
    S->>DB: UPDATE emp_st_cd=APRVD, aprv_dt=NOW()
    S->>DB: INSERT ptl.th_user_actn (감사로그)
    DB-->>S: OK
    S-->>W: 200
    W-->>MOL: ✓ 승인 완료
```

## SEQ-06: 로그인 + 5회 실패 자동 잠금

```mermaid
sequenceDiagram
    autonumber
    actor U as 사용자
    participant W as Login Page
    participant SF as SecurityFilter
    participant AP as IcasAuthenticationProvider
    participant AM as AuthMapper
    participant DB as DB

    U->>W: userId + password 입력
    W->>SF: POST /login (UsernamePasswordAuth)
    SF->>AP: authenticate()
    AP->>AM: selectUserAuthInfo(userId)
    AM->>DB: SELECT user + ognz JOIN

    alt 비밀번호 불일치
        AP->>AM: incrementPasswordFailCount
        AM->>DB: UPDATE pswd_fail_cnt+1, acnt_lock_yn=(IF count>=5 'Y')
        AP-->>W: BadCredentialsException
        W-->>U: ❌ 비밀번호 오류 (남은 시도 N회)
    else 5회 초과
        AP-->>W: LockedException
        W-->>U: 🔒 계정 잠김 (관리자 문의)
    else 성공
        AP->>AM: resetPasswordFailCount + updateLastLogn
        AP->>AM: selectPrgrmPathsInq/Inpt + selectRoleIds
        AP-->>W: IcasUser (권한 매트릭스 포함)
        W-->>U: ✓ 메인 대시보드 진입
    end
```

## 4. 화면 패턴 (대표 5종)

### 4.1 목록 화면 (EMP/ER/VR/CEF/EUCR/OoM/CORSIA 공용)
```
[페이지 헤더: 화면명 + 브레드크럼 │ 신규 등록 버튼]
[검색 카드: 보고연도·운영사·상태 + 조회·초기화]
[그리드 카드:
  - 총 N건 + 엑셀 다운로드
  - 헤더(다크 네이비) + 행(호버 라이트블루) + 액션 컬럼
  - 페이지네이션
]
```

### 4.2 상세 화면 (EMP/ER/VR/CEF/EUCR/OoM 공용)
```
[브레드크럼 + "목록" + "재계산" 버튼]
[기본정보 카드: ID·운영사·보고연도·버전·상태·핵심 KPI]
[라이프사이클 스테퍼: 작성중→제출→검토중→권고→승인→취소]
[액션 버튼 바: 권한·상태에 따라 동적 노출]
[탭 영역: 도메인별 4~8 탭]
```

### 4.3 대시보드 (메인·SAF 이행·통계)
```
[KPI 카드 4종 (그리드 4컬럼)]
[차트 영역 (ECharts) + 표]
[빠른 메뉴 사이드 패널]
```

### 4.4 폼 화면 (비밀번호 변경·ICAO 송신)
```
[정책/안내 카드]
[폼 필드 (검증 인라인)]
[액션 버튼 (제출·취소)]
```

### 4.5 매트릭스 (워크플로우·검증배정·시스템 상태)
```
[필터 카드]
[2차원 매트릭스 그리드 (운영사 × 도메인 / 권한 × 화면 등)]
```

## 5. 모바일 대응 (1차년도 가정)

- 데스크탑 1280px+ 최적화
- 800~1280px: 사이드바 자동 접힘 (2차년도 작업 예정)
- 800px 미만: 햄버거 메뉴 토글 (2차년도)

## 6. 접근성

- 시맨틱 HTML (header/nav/main/aside/footer)
- ARIA 레이블 적용 (aria-label, role)
- 키보드 네비게이션 (Tab/Enter/Esc)
- 색상 대비 ≥ 4.5:1 (KWCAG 2.1 AA)

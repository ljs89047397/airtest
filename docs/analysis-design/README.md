## 📊 시스템 현황 요약 (산출 자료)

### 기능
- RFP 11박스 9 완료 + 2 placeholder (2차년도)
- 화면 36개 (사이드바 라우트)
- 유즈케이스 32개 (4 actor × 8 시나리오)
- DB 81 테이블 (6 스키마)

### 테스트
- E2E 8 시나리오 243건 100% 통과
- UAT 32 케이스 (인수 시점 검증)

### 권한
- 4 actor × 8 시스템 권한 = 32 사용자 그룹
- RBAC 3단계 (사용자→역할→권한→프로그램)
- AuthorityInterceptor + DataScopeValidator 다층 검증

### 보안
- 5회 실패 자동 잠금
- 비밀번호 정책 (8자+영문+숫자+특수문자)
- 세션 30분 idle 만료 + 25분 경고
- CSRF + XSS 보호

### 시행계획 대비
- CORSIA 운영 일정 캘린더 (5~7월 검증, 7월 ICAO 제출, 11월 통보)
- SAF 공급의무/급유의무 분리 (시행계획 p.5~6)
- 친환경 항공기 도입 트래커 (시행계획 p.8, 5사 40대)
- 감축수단별 기여도 차트 (시행계획 p.7, SAF 4%·친환경 5%·운항 1%)
- ICAO 송신 콘솔 Mock (시행계획 p.13)
